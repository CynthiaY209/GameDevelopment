## 作业说明：

 + 对角色进行替换

 + 角色控制及动画：
 
     + WASD键行走
  
     + 在行走基础上按B，切换为奔跑
  
     + J键攻击
     
     * 走/跑过程中按J，会先停下完成攻击动作，判断动画播放结束后，继续走/跑
     
 + 击中火花：
 
     + 角色攻击时，剑击中石头，接触点可产生火花粒子效果
  
     + 非攻击状态下，角色的剑触碰到石头，不会产生火花
  
 + 相机：
 
     + 上帝视角跟随角色
  
     + 使用cinemachine，击中石头时产生震动效果，振幅频率可调
